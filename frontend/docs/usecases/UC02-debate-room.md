# UC02: 토론방

## 개요
토론 주제를 기반으로 토론방을 생성, 검색, 참여, 퇴장, 종료하는 기능을 제공한다.

---

## UC02-1: 토론방 생성

### 액터
- 주 액터: 인증된 사용자

### 사전 조건
- 사용자가 로그인된 상태

### 기본 흐름
1. 홈 페이지에서 "새 토론" 버튼을 클릭한다.
2. 토론방 생성 모달이 열린다.
3. 토론 주제(필수), 설명, 태그를 입력/선택한다.
4. "생성" 버튼을 클릭한다.
5. 토론방이 생성되고, 생성자는 NEUTRAL 입장으로 자동 참여한다.
6. 생성된 토론방 상세 페이지로 이동한다.

### 사후 조건
- 새로운 DebateRoom 엔티티가 OPEN 상태로 생성됨
- 생성자가 RoomParticipant로 등록됨

### UI 매핑
- 화면: `CreateRoomModal`
- 관련 API: `POST /api/rooms`

---

## UC02-2: 토론방 검색/필터

### 액터
- 주 액터: 모든 사용자 (인증 불필요)

### 기본 흐름
1. 홈 페이지(`/`)에서 토론방 목록이 카드 형태로 표시된다.
2. 검색창에 키워드를 입력하여 제목/설명 기반 검색을 한다.
3. 상태 필터 (전체/진행중/종료)를 선택한다.
4. 태그 칩을 클릭하여 태그별 필터링을 한다.
5. 결과가 실시간으로 갱신된다.
6. 페이지네이션으로 추가 결과를 확인한다.

### UI 매핑
- 화면: `HomePage`, `RoomFilters`, `RoomCard`
- 관련 API: `GET /api/rooms?page=&size=&q=&tag=&status=`

---

## UC02-3: 토론방 입장

### 액터
- 주 액터: 인증된 사용자

### 사전 조건
- 토론방이 OPEN 상태
- 사용자가 해당 토론방에 아직 참여하지 않은 상태

### 기본 흐름
1. 토론방 상세 페이지에서 "참여하기" 버튼을 클릭한다.
2. 입장 선택 모달이 표시된다 (찬성/반대/중립).
3. 원하는 입장을 선택한다.
4. 참여자 목록에 사용자가 추가된다.
5. 채팅 입력이 활성화된다.

### 사후 조건
- RoomParticipant 엔티티가 생성됨
- 토론방 참여자 수가 갱신됨

### UI 매핑
- 화면: `RoomHeader` (참여하기 버튼 + 모달)
- 관련 API: `POST /api/rooms/{roomId}/participants`

---

## UC02-4: 토론방 퇴장

### 기본 흐름
1. 토론방 상세 페이지에서 "나가기" 버튼을 클릭한다.
2. 참여자 목록에서 사용자가 제거된다.
3. 채팅 입력이 비활성화된다.

### 관련 API: `DELETE /api/rooms/{roomId}/participants/me`

---

## UC02-5: 토론방 종료

### 액터
- 주 액터: 방장 (토론방 생성자)

### 기본 흐름
1. 방장이 "토론 종료" 버튼을 클릭한다.
2. 확인 다이얼로그가 표시된다.
3. 확인 시 토론방 상태가 CLOSED로 변경된다.
4. 새 메시지, 논거 추가가 비활성화된다.

### 관련 API: `POST /api/rooms/{roomId}/close`
