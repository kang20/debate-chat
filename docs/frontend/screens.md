# 화면 구성

## 1. 메인 화면 (HomePage)

### 1.1 헤더 (Header)
- **좌측**: 로고(MessageSquare 아이콘 + "토론챗") → 클릭 시 홈으로 이동
- **우측 (로그인 전)**: "로그인" 고스트 버튼 + "회원가입" 프라이머리 버튼
- **우측 (로그인 후)**:
  - "새 토론" 버튼(Plus 아이콘)
  - 알림 벨 아이콘 + 읽지 않은 알림 수 빨간 배지(9개 초과 시 "9+")
  - 알림 팝오버: 벨 클릭 시 최근 5개 알림 드롭다운, "모두 읽음" 버튼, "모든 알림 보기" 링크
  - 프로필 영역: 아바타 + 닉네임 표시, 클릭 시 프로필로 이동
  - 로그아웃 버튼(LogOut 아이콘)
- **모바일**: 햄버거 메뉴(☰) → 열면 프로필/알림/로그아웃 세로 메뉴

### 1.2 히어로 섹션
- **좌측**:
  - 실시간 토론 N개 진행 중 배지 (인디고 라운드 배지, 펄스 도트)
  - 메인 카피: "생각이 부딪치는 곳, **한 줄의 의견**으로 시작하세요."
  - 서브 카피: "찬성과 반대, 그 사이의 모든 입장. 토론챗에서는 모든 메시지에 사이드가 새겨집니다."
  - 검색 입력창(돋보기 아이콘) + "새 토론 시작" 버튼 (로그인 시)
- **우측** (데스크톱만): 통계 카드 3개
  - 진행 중인 토론 수
  - 활동 중 참여자 수
  - 오늘의 메시지 수

### 1.3 관심 분야 카테고리 바
- 카테고리 버튼 7개 가로 그리드: 기술, 정책, 경제, 사회, 직장, 복지, 라이프스타일
- 각 버튼에 태그명(#기술) + 설명(AI · 데이터 · 인터넷)
- 클릭 시 해당 태그로 필터링, 다시 클릭 시 해제
- 활성 상태: 인디고 보더 + 인디고 배경
- "전체 보기" 버튼으로 필터 초기화

### 1.4 필터 & 정렬 툴바
- **상태 필터**: 전체 / 진행 중 / 종료 (세그먼트 컨트롤 스타일)
  - 활성: 검정 배경 + 흰 텍스트
- **정렬**: 인기 / 최신 / 박빙 (세그먼트 컨트롤)
  - 활성: 인디고 배경 + 인디고 텍스트
  - 인기: 참여자+찬반 수 합산 내림차순
  - 최신: 생성일 내림차순
  - 박빙: 찬반 차이 절대값 오름차순

### 1.5 Featured 카드 (지금 뜨거운 토론)
- 참여자가 가장 많은 진행 중 토론방 1개를 강조 표시
- **좌측**: "실시간" 펄스 배지 + 태그, 제목(2xl), 설명, 작성자 아바타 + 닉네임 + 상대시간, 참여자 수
- **우측**: 현재 분포(찬성 우세/반대 우세/박빙), 스탠스 바, 찬성/중립/반대 퍼센트+인원 카드, "토론 참여하기" 버튼
- 상태 필터 활성 시 숨김

### 1.6 토론방 카드 그리드
- 3열 그리드(lg), 2열(md), 1열(sm)
- **진행 중인 토론** 섹션 + **종료된 토론** 섹션으로 분리
- 각 카드:
  - 상단: 태그(최대 2개) + 상대시간(또는 "종료" 배지)
  - 제목(15px, bold, 2줄 clamp)
  - 설명(13px, 2줄 clamp)
  - 찬성/중립/반대 퍼센트 + 스탠스 바
  - 하단: 작성자 아바타+닉네임 + 참여자 수
  - 호버: 약간 위로 이동(-translate-y-0.5) + 그림자 증가
- 검색 결과 없을 시: 빈 상태 메시지("검색 결과가 없습니다")

### 1.7 토론방 생성 모달
- 모달(중앙 오버레이) 방식으로 표시
- 입력 필드:
  - 토론 주제 (필수, 최대 100자)
  - 설명 (textarea, 3줄)
  - 태그 (입력+Enter 또는 자주 쓰는 태그 클릭, 추가된 태그는 인디고 칩으로 표시)
  - 공개 범위 (공개/비공개 선택 카드, 비공개 시 자물쇠 아이콘)
- 하단: "취소" + "토론방 생성" 버튼
- 생성 성공 시 토스트 알림 + 해당 방으로 이동

## 2. 토론방 상세 (RoomDetailPage)

### 2.1 방 헤더 (RoomHeader)
- 뒤로가기 버튼(←) → 홈으로 이동
- 토론 주제(제목) + 상태 배지("종료") + 비공개 시 자물쇠 아이콘
- 설명 텍스트
- 태그 배지 목록
- 통계: 참여자 수, 찬성/반대/중립 인원(색상 코딩), 생성 상대시간
- **비참여자**: "참여하기" 버튼 → 입장 선택 모달(찬성/반대/중립 3개 버튼)
- **참여자**: "나가기" 버튼
- **방장**: "토론 종료" 위험(danger) 버튼 추가

### 2.2 스탠스 요약 바
- 찬성 N / 중립 N / 반대 N 텍스트
- 파란색-빨간색 수평 비율 바

### 2.3 탭 전환
- 필 형태(pill) 탭 스위처: **실시간 채팅**(MessageCircle 아이콘) | **논거 트리**(TreePine 아이콘)
- 활성 탭: 인디고 배경 + 흰 텍스트

### 2.4 채팅 탭 (ChatPanel)
- 고정 높이(500px) 메시지 영역 + 자동 스크롤
- 상단 "이전 메시지 불러오기" 버튼 (더 있을 경우)
- 하단: textarea 입력창 + 전송 버튼(Send 아이콘)
  - Enter로 전송, Shift+Enter로 줄바꿈
  - 비참여자: "토론에 참여해야 메시지를 보낼 수 있습니다" 안내 텍스트

### 2.5 논거 트리 탭 (ArgumentTree)
- "논거 트리" 제목 + "논거 추가" 버튼 (참여자만)
- 논거가 없을 경우: TreePine 아이콘 + 빈 상태 메시지
- 트리 구조 재귀 렌더링 (ArgumentNode)

### 2.6 참여자 사이드바 (ParticipantSidebar, 데스크톱만)
- "참여자 (N)" 제목
- 입장별 그룹: 찬성 → 반대 → 중립 순서
- 각 그룹: 입장 배지(SideBadge) + 인원 수 + 아바타/닉네임 목록
- 해당 입장에 참여자 없으면 빈 상태 메시지

## 3. 인증 화면

공통 레이아웃: 화면 중앙 정렬, 그레이 배경, 흰색 카드(max-w-md, rounded-2xl, border, shadow)

### 3.1 로그인
- 상단: 로고(MessageSquare 아이콘 + "토론챗") — 인디고 색상
- 제목: "다시 오신 것을 환영합니다"
- 설명: "계정에 로그인하여 토론에 참여하세요"
- 입력 필드: 이메일 + 비밀번호
- "로그인" 버튼 (전체 너비, 로딩 시 "로그인 중...")
- 하단 링크: "계정이 없으신가요? **회원가입**"
- 테스트 계정 안내 박스: 회색 배경 카드에 이메일/비밀번호 표시

### 3.2 회원가입
- 상단: 로고(MessageSquare 아이콘 + "토론챗")
- 제목: "회원가입"
- 설명: "새 계정을 만들고 토론에 참여하세요"
- 입력 필드: 이메일 + 닉네임(3-20자) + 비밀번호(8자 이상)
- 각 필드 아래 유효성 검사 에러 메시지(빨간색)
- "가입하기" 버튼 (전체 너비, 로딩 시 "가입 중...")
- 하단 링크: "이미 계정이 있으신가요? **로그인**"

## 4. 프로필 화면
- 아바타 + 닉네임 + 이메일 + 자기소개 + 가입일
- 편집 모드 전환 가능

## 5. 알림 화면
- 상단: "모두 읽음" 버튼
- 알림 카드 리스트: 유형 아이콘 + 발신자 + 내용 요약 + 시간 + 읽음 상태
